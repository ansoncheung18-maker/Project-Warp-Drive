## 2. Dyson, F.J. (1960) — Dyson Swarm 原始概念

### 2.1 文獻基本資料

| 項目 | 內容 |
|:---|:---|
| **作者** | Dyson, F.J. |
| **年份** | 1960 |
| **標題** | "Search for Artificial Stellar Sources of Infrared Radiation" |
| **期刊** | *Science*, 131(3414), 1667-1668 |
| **被引用次數** | 3,000+ |
| **機構** | Institute for Advanced Study, Princeton |

### 2.2 論文背景與動機

| 項目 | 內容 |
|:---|:---|
| **科學問題** | 高等文明會點樣獲取能量？ |
| **研究目標** | 提出一個可以搜尋高等文明嘅方法（Dyson Swarm） |
| **重要性** | 恆星級能源概念嘅起源 |

### 2.3 核心概念

| 概念 | 解釋 |
|:---|:---|
| **Dyson Sphere** | 包圍恆星嘅球體，捕捉所有能量 |
| **Dyson Swarm** | 由大量獨立衛星組成嘅「球體」 |
| **紅外線特徵** | Dyson Swarm 會發出紅外線，可以用嚟搜尋 |
| **恆星級能源** | 捕捉恆星嘅全部能量輸出 |

### 2.4 關鍵結果

| 指標 | 數值 | 意義 |
|:---|:---|:---|
| **結果** | 提出 Dyson Swarm 概念 | 恆星級能源概念嘅起源 |
| **結論** | 高等文明可能會建造 Dyson Swarm | 為你嘅 Project 提供能源概念 |
| **影響** | 啟發後續所有 Dyson Swarm 研究 | 你引用嘅所有 Dyson Swarm 研究都源於呢篇論文 |

### 2.5 對你 Project 嘅支持

| 支持點 | 解釋 |
|:---|:---|
| **Dyson Swarm 係真實概念** | Dyson (1960) 提出咗呢個概念，已經存在超過 60 年 |
| **能源來源** | 你嘅 Project 使用 Dyson Swarm 作為能源來源 |
| **科學界認可** | Dyson Swarm 已經被廣泛討論同研究 |

### 2.6 局限性同解決方案

| 局限性 | 解決方案 |
|:---|:---|
| 純理論概念，未經建造 | 你嘅 Project 係基於 Dyson Swarm 概念設計能源系統 |
| 建造難度極高 | 你嘅 Project 結合自複製機械人同 ISRU 技術 |
| 成本極高 | 你嘅 Project 係概念設計階段，未考慮實際建造 |
