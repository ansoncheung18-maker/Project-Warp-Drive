## 1. Lamoreaux, S.K. (1997)

### 1.1 文獻基本資料

| 項目 | 內容 |
|:---|:---|
| **作者** | Lamoreaux, S.K. |
| **年份** | 1997 |
| **標題** | "Demonstration of the Casimir Force in the 0.6 to 6 μm Range" |
| **期刊** | *Physical Review Letters*, 78(1), 5-8 |
| **被引用次數** | 1,500+ |
| **實驗地點** | Los Alamos National Laboratory |

### 1.2 實驗背景與動機

| 項目 | 內容 |
|:---|:---|
| **科學問題** | Casimir 效應自 1948 年被預測以來，從未被精確測量過 |
| **實驗目標** | 首次精確測量 Casimir 效應，驗證理論預測 |
| **實驗難度** | 需要測量極細微嘅力（~10⁻⁷ N） |

### 1.3 實驗方法

| 項目 | 內容 |
|:---|:---|
| **實驗裝置** | 扭擺 (Torsion Pendulum) |
| **測量對象** | 兩塊金屬板之間嘅吸引力 |
| **板間距** | 0.6 – 6 μm |
| **測量精度** | < 5% |
| **環境** | 高真空 (~10⁻⁶ Torr) |

### 1.4 關鍵結果

| 指標 | 數值 | 意義 |
|:---|:---|:---|
| **測量誤差** | < 5% | 首次高精度驗證 Casimir 效應 |
| **結果** | 實驗數據與理論預測一致 | Casimir 效應係真實物理現象 |
| **結論** | Casimir 效應存在，可以產生負能量 | 為你嘅 Project 提供物理基礎 |

### 1.5 對你 Project 嘅支持

| 支持點 | 解釋 |
|:---|:---|
| **Casimir 效應係真實嘅** | Lamoreaux 證明咗 Casimir 效應唔係理論推測，而係真實物理現象 |
| **可以產生負能量** | Casimir 效應嘅本質就係負能量密度 |

## 2. Mohideen, U. & Roy, A. (1998)

### 2.1 文獻基本資料

| 項目 | 內容 |
|:---|:---|
| **作者** | Mohideen, U. & Roy, A. |
| **年份** | 1998 |
| **標題** | "Precision Measurement of the Casimir Force" |
| **期刊** | *Physical Review Letters*, 81(21), 4549-4552 |
| **被引用次數** | 1,000+ |
| **實驗地點** | University of California, Riverside |

### 2.2 實驗背景與動機

| 項目 | 內容 |
|:---|:---|
| **科學問題** | Lamoreaux 嘅測量雖然成功，但精度可以更高 |
| **實驗目標** | 用原子力顯微鏡 (AFM) 更精確測量 Casimir 效應 |
| **實驗難度** | AFM 對環境震動極敏感 |

### 2.3 實驗方法

| 項目 | 內容 |
|:---|:---|
| **實驗裝置** | 原子力顯微鏡 (AFM) |
| **測量對象** | 金屬球體同平板之間嘅 Casimir 力 |
| **板間距** | 100 nm – 1 μm |
| **測量精度** | < 1% |
| **環境** | 高真空 |

### 2.4 關鍵結果

| 指標 | 數值 | 意義 |
|:---|:---|:---|
| **測量誤差** | < 1% | 比 Lamoreaux 更精確 |
| **結果** | 實驗數據與理論預測一致 | Casimir 效應嘅可信度進一步提升 |
| **新發現** | AFM 係測量 Casimir 效應嘅有效工具 | 為未來實驗提供新方法 |
