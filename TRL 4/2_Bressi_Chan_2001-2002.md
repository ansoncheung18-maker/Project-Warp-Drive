## 1. Bressi, G. et al. (2002)

### 1.1 文獻基本資料

| 項目 | 內容 |
|:---|:---|
| **作者** | Bressi, G., Carugno, G., Onofrio, R., & Ruoso, G. |
| **年份** | 2002 |
| **標題** | "Measurement of the Casimir Force between Parallel Metallic Surfaces" |
| **期刊** | *Physical Review Letters*, 88(4), 041804 |
| **被引用次數** | 800+ |
| **實驗地點** | University of Padua, Italy |

### 1.2 實驗背景與動機

| 項目 | 內容 |
|:---|:---|
| **科學問題** | Lamoreaux (1997) 同 Mohideen & Roy (1998) 雖然成功測量 Casimir 效應，但誤差仍然可以進一步降低 |
| **實驗目標** | 用平行金屬板直接測量 Casimir 效應，達到最高精度 |
| **實驗難度** | 需要保持兩塊平行板之間嘅極小間距，而且需要極高穩定性 |

### 1.3 實驗方法

| 項目 | 內容 |
|:---|:---|
| **實驗裝置** | 平行板測量系統 |
| **測量對象** | 兩塊平行金屬板之間嘅 Casimir 力 |
| **板間距** | 0.5 – 3 μm |
| **測量精度** | **< 1%** |
| **環境** | 高真空，溫度穩定 |

### 1.4 關鍵結果

| 指標 | 數值 | 意義 |
|:---|:---|:---|
| **測量誤差** | < 1% | 當時最高精度嘅 Casimir 效應測量 |
| **結果** | 實驗數據與理論預測一致 | Casimir 效應嘅可信度進一步提升 |
| **結論** | Casimir 效應可以精確測量，誤差 < 1% | 為你嘅 Project 提供可靠嘅物理基礎 |

### 1.5 對你 Project 嘅支持

| 支持點 | 解釋 |
|:---|:---|
| **Casimir 效應測量係可靠嘅** | 誤差 < 1%，證明 Casimir 效應唔係偶然現象 |
| **理論同實驗一致** | 證明你嘅數學模型可以有信心 |
| **技術成熟** | 平行板測量係成熟技術 |

### 1.6 局限性同解決方案

| 局限性 | 解決方案 |
|:---|:---|
| 板間距只做到 0.5 μm | 現代 MEMS 技術可以做到 < 10 nm |
| 只測量吸引力 | Munday et al. (2009) 已經測量到排斥力 |
| 只喺地球環境測試 | 太空環境（靈神星）更適合 Casimir 效應運作 |

| **環境** | 真空 |
