## 2. 核心問題：點樣製造一個曲率泡？

### 2.1 曲率泡嘅「材料」係咩？

| 傳統飛船 | 曲率泡飛船 |
|:---|:---|
| 需要燃料 | **需要「負能量」** |
| 需要引擎 | **需要「Casimir 陣列」** |
| 需要結構 | **需要「場腔固定技術」** |
| 需要控制系統 | **需要「AI 動態場反饋」** |
| 需要油箱 | **需要「超導儲能環」** |

**關鍵**：曲率泡嘅「材料」唔係金屬或燃料，而係 **「負能量場態」**，而且航行期間需要 **持續消耗能量** 維持曲率泡。

### 2.2 製造曲率泡嘅 6 個步驟 (更新)

| 步驟 | 動作 | 產出 |
|:---|:---|:---|
| **1. 產生負能量** | 用 Casimir 陣列產生負能量密度 | 微觀負能量 |
| **2. 放大負能量** | 用 1 km³ 三維陣列放大 | 宏觀負能量 |
| **3. 儲存負能量** | 用超導儲能環儲存 | 可控制釋放 |
| **4. 釋放負能量** | 用場釋放閥控制流量 | 可控能量流 |
| **5. 形成曲率泡** | 用 Alcubierre 度規扭曲時空 | 曲率泡 |
| **6. 維持曲率泡** | 用 AI 動態場反饋實時補償 | 可持續曲率泡 |


## 3. 步驟 1：產生負能量 — Casimir 陣列

### 3.1 Casimir 效應原理

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 原理 | 兩塊金屬板之間嘅量子漲落產生負能量密度 | 1948 年由 Casimir 預測 |
| 板間距 | < 10 nm | 距離越近，負能量密度越高 |
| 負能量密度 | ~10⁻⁶ J/m² | 極細，需要放大 |

### 3.2 Casimir 陣列設計 (1km³)

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 陣列規模 | 1 km × 1 km × 1 km | 三維立方體 |
| 金屬板數量 | **~3.7 × 10¹⁰ 塊** | 每塊 1m × 1m × 1nm |
| 金屬板材料 | 金/銀/銅 | 高導電性 |
| 板間距 | **0.5 nm** | ±0.01 nm 精度 |
| 總質量 | ~1,850 噸 | |
| 每秒負能量產出 | **~1.66 × 10³⁶ J/s** | 比目標多 10²² 倍 |

### 3.3 製造 Casimir 陣列嘅挑戰

| 挑戰 | 解決方案 |
|:---|:---|
| 納米級精度 | 用 R1/R2 機械人 + 原子力顯微鏡 (AFM) 控制 |
| 熱膨脹 | 喺靈神星低溫環境 (~3K) 運作 |
| 量子漲落干擾 | 用超導場腔屏蔽外部干擾 |


## 4. 步驟 2：儲存負能量 — 超導儲能環 (新增)

### 4.1 點解需要儲存？

| 問題 | 解釋 |
|:---|:---|
| 航行期間需要持續能量 | 曲率泡會消耗能量維持時空扭曲 |
| 唔可以「邊飛邊整」 | 曲率泡內部時空扭曲會干擾 Casimir 效應 |
| 預先儲存更穩定 | 航行期間關閉 Casimir 陣列，完全依靠儲存環 |

### 4.2 儲存環設計

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 儲存方式 | 超導儲能環 | 類似 LHC |
| 儲存環直徑 | 2 km | |

| 儲存效率 | **> 99%** | 超導零電阻 |
| 總質量 | ~100 噸 | |
