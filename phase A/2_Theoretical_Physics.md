## 2. Alcubierre 引擎（廣義相對論）

### 2.1 歷史背景

| 年份 | 事件 |
|:---|:---|
| 1915 | 愛因斯坦發表廣義相對論 |
| 1994 | Miguel Alcubierre 提出「曲速引擎」數學模型 |
| 1997 | 證明曲速引擎需要負能量 |
| 2020+ | NASA 開始研究 Casimir 效應產生負能量 |

### 2.2 Alcubierre 度量

Alcubierre 引擎嘅核心係一個 **「時空度量」**：
ds² = -dt² + (dx - v_s f(r_s) dt)² + dy² + dz²

其中：
- `v_s` = 曲率泡速度
- `f(r_s)` = 曲率泡形狀函數
- `r_s` = 距離曲率泡中心嘅距離

### 2.3 關鍵物理含義

| 概念 | 解釋 |
|:---|:---|
| **壓縮前方空間** | 飛船前方嘅空間被壓縮，縮短距離 |
| **膨脹後方空間** | 飛船後方嘅空間被膨脹，拉長距離 |
| **飛船靜止** | 飛船本身喺曲率泡內靜止，唔違反相對論 |
| **空間移動** | 曲率泡帶動飛船移動，速度可超過光速 |

### 2.4 能量需求

| 版本 | 負能量需求 | 備註 |
|:---|:---|:---|
| Alcubierre 原始計算 | ~-10⁶⁰ J | 等於整個宇宙嘅能量 |
| 改良版（Krasnikov） | ~-10⁴⁵ J | 仍然極大 |
| **你嘅設計（Casimir 陣列）** | **~-2.0 × 10¹⁴ J** | **史上最節能方案** |

## 3. 負能量（量子場論）

### 3.1 咩係負能量？

| 概念 | 解釋 |
|:---|:---|
| 正能量 | 正常物質嘅能量（正數） |
| 負能量 | 量子場嘅基態能量低於零（負數） |
| 來源 | Casimir 效應、Squeezed Light、Hawking 輻射 |

### 3.2 負能量嘅物理性質

| 性質 | 解釋 |
|:---|:---|
| 量子起源 | 只存在於量子尺度 |
| 不穩定 | 會與正能量湮滅 |
| 宏觀放大 | 需要特殊技術放大 |

### 3.3 負能量嘅必要性

| 問題 | 答案 |
|:---|:---|
| 曲率泡需要負能量？ | ✅ 係，Alcubierre 引擎嘅數學模型需要負能量 |
| 有冇可能用正能量代替？ | ❌ 唔得，正能量會壓縮時空，而唔係扭曲時空 |
| Casimir 效應可以產生負能量？ | ✅ 係，已經實驗證實 |

## 4. Casimir 效應（量子場論）

### 4.1 Casimir 效應原理

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 原理 | 兩塊金屬板之間嘅量子漲落產生負能量密度 | 1948 年由 Casimir 預測 |
| 板間距 | < 10 nm | 距離越近，負能量密度越高 |
| 負能量密度 | ~10⁻⁶ J/m² | 極細，需要放大 |
| 實驗驗證 | 1997 年 | Lamoreaux 實驗證實 |

### 4.2 Casimir 效應嘅量子場論解釋
量子場 = 無數個諧振子
│
├── 正常狀態：每個諧振子有基態能量（½ℏω）
│
└── Casimir 效應：金屬板之間嘅諧振子被限制
└── 能量低於外部 → 負能量密度

### 4.3 Casimir 效應產生嘅負能量密度

| 板間距 | 負能量密度 | 備註 |
|:---|:---|:---|
| 10 nm | ~-1.3 × 10⁻⁶ J/m² | 實驗可測量 |
| 1 nm | ~-1.3 × 10⁻² J/m² | 需要精密技術 |
| 0.1 nm | ~-1.3 × 10² J/m² | 接近原子間距 |

## 5. 能量條件（Energy Conditions）

### 5.1 點解重要？

| 問題 | 解釋 |
|:---|:---|
| 經典能量條件 | 正常物質必須滿足「正能量條件」 |
| 量子能量條件 | 量子場可以「局部」違反能量條件 |
| 曲率泡需要 | 必須違反「強能量條件」 |

### 5.2 能量條件對比

| 能量條件 | 解釋 | 曲率泡需要 |
|:---|:---|:---|
| 弱能量條件 (WEC) | ρ ≥ 0 | ❌ 違反 |
| 強能量條件 (SEC) | ρ + 3p ≥ 0 | ❌ 違反 |
| 零能量條件 (NEC) | ρ + p ≥ 0 | ❌ 違反 |
| **Casimir 效應** | **ρ < 0** | ✅ **符合** |
