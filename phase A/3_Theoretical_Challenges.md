## 2. 挑戰 1：因果律（時間旅行悖論）

### 2.1 問題描述

| 問題 | 解釋 |
|:---|:---|
| 超光速飛行 | 可能導致「訊息傳播速度 > 光速」 |
| 相對論效應 | 喺某啲參考系中，超光速訊號可能「向後傳播」 |
| 悖論 | 可能導致「殺死自己祖先」之類嘅時間旅行悖論 |

### 2.2 解決方案：Novikov 自洽性原則

| 原則 | 解釋 |
|:---|:---|
| **Novikov 自洽性原則** | 時間旅行嘅結果必須「自洽」— 你殺唔到自己祖先 |
| **物理定律保護** | 任何試圖違反因果律嘅事件都會被物理定律阻止 |
| **量子力學解釋** | 量子力學可能隱藏因果律保護機制 |

### 2.3 工程含義

| 結論 | 解釋 |
|:---|:---|
| 因果律問題係「理論討論」 | 唔係工程障礙 |
| 曲率泡本身唔會產生悖論 | Alcubierre 引擎嘅數學模型係自洽嘅 |
| 唔需要額外工程措施 | 因果律問題由物理定律自動處理 |
