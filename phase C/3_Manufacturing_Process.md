## 3. M-01：礦石開採 (R1-Psyche)

### 3.1 工序說明

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 負責機械人 | R1-Psyche | 開採者 |
| 數量 | 100 部 | 同時工作 |
| 工作地點 | 靈神星表面 | 金屬含量高嘅區域 |
| 開採方式 | 激光切割 + 真空吸塵 | |
| 產出 | 金、銀、銅、鐵、鎳礦石 | |
| 時間 | **18.5 日** | |

### 3.2 詳細步驟

| 步驟 | 動作 | 說明 |
|:---|:---|:---|
| 1 | 礦床定位 | 用伽馬射線探測器搵出金屬含量高嘅區域 |
| 2 | 激光切割 | 用 1kW 光纖激光切割金屬礦石 |
| 3 | 碎屑收集 | 用真空吸塵 + 靜電板收集礦石碎屑 |
| 4 | 分類儲存 | 將不同金屬分類放入收集箱 |
| 5 | 運輸到加工區 | 將礦石運到 R2 加工區 |

### 3.3 產出規格

| 產出 | 數量 | 純度 | 備註 |
|:---|:---|:---|:---|
| 金礦石 | 1,850 噸 | ~90% | 用於 Casimir 金屬板 |
| 鐵礦石 | 600 噸 | ~85% | 用於因瓦合金 |
| 鎳礦石 | 100 噸 | ~80% | 用於因瓦合金 |
| 銅礦石 | 50 噸 | ~90% | 用於冷卻管道 |

## 5. M-03：金板製造 (R2-Psyche)

### 5.1 工序說明

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 負責機械人 | R2-Psyche | 加工者 |
| 數量 | 300 部 | 專門負責金板 |
| 工作地點 | 靈神星表面工廠 | 真空環境 |
| 加工精度 | 0.01 nm | 原子級 |
| 產出 | 1m × 1m × 1nm 金板 | |
| 數量 | 3.7 × 10¹⁰ 塊 | |
| 時間 | **20 日** | |

### 5.2 詳細步驟

| 步驟 | 動作 | 說明 |
|:---|:---|:---|
| 1 | 金錠加熱 | 將金錠加熱到接近熔點 |
| 2 | 納米軋製 | 用納米軋機壓延成 1nm 厚度 |
| 3 | 尺寸切割 | 切割成 1m × 1m 正方形 |
| 4 | AFM 拋光 | 用原子力顯微鏡拋光表面，達原子級光滑 |
| 5 | 厚度檢測 | 用激光干涉儀檢測厚度 (1nm ± 0.01nm) |
| 6 | 表面檢測 | 用 AFM 檢測表面粗糙度 (< 0.01nm) |
| 7 | 包裝儲存 | 將合格金板放入儲存箱 |

### 5.3 產出規格

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 尺寸 | 1m × 1m × 1nm | |
| 表面粗糙度 | < 0.01 nm | 原子級光滑 |
| 厚度公差 | ±0.01 nm | |
| 數量 | 3.7 × 10¹⁰ 塊 | |
| 總質量 | 1,850 噸 | |
