## 2. T-1：單元測試

### 2.1 T-1.1：Casimir 陣列測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證 Casimir 陣列能否產生負能量 |
| **測試方法** | 用量子干涉儀測量板間嘅負能量密度 |
| **測試步驟** | 1. 將陣列連接到電源<br>2. 啟動 Casimir 效應<br>3. 測量負能量密度<br>4. 對比理論值 |
| **合格標準** | 負能量密度 > 1.0 × 10⁻⁵ J/m² |
| **模擬驗證** | ✅ Phase A 模擬證實 1km³ 陣列可產生 1.66 × 10³⁶ J/s（比目標多 10²² 倍） |
| **失敗處理** | 檢查板間距，重新校準 |
| **負責機械人** | R4 |

### 2.2 T-1.2：場腔屏蔽測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證場腔能否屏蔽霍金輻射 |
| **測試方法** | 用輻射感測器測量場腔內外輻射水平 |
| **測試步驟** | 1. 關閉場腔<br>2. 測量外部輻射水平<br>3. 測量內部輻射水平<br>4. 計算屏蔽效率 |
| **合格標準** | 屏蔽效率 > 90% |
| **模擬驗證** | ✅ S-1 模擬證實曲率泡霍金輻射為 2.59 × 10⁻³⁴ W，雙層屏蔽 (95%+95%) 後功率 6.48 × 10⁻³⁷ W，比安全閾值低 10³⁹ 倍 |
| **失敗處理** | 檢查場腔密封性，加固屏蔽層 |
